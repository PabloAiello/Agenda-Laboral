# 📅 Agenda Laboral

Una aplicación web desarrollada para organizar tareas, reuniones, recordatorios y actividades laborales.

## 🚀 Funcionalidades

- ✅ Crear tareas laborales.
- 📅 Asignar fecha y hora.
- 📝 Agregar descripciones.
- 🔔 Alertas de próximos vencimientos.
- ✔️ Marcar tareas como completadas.
- 🗑️ Eliminar tareas.
- 💾 Guardado automático en el navegador (LocalStorage).
- 📱 Diseño adaptable para PC, tablet y celular.

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Estructura del proyecto

```
Agenda-Laboral/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## 🎯 Objetivo

Facilitar la organización diaria de tareas laborales mediante una interfaz simple, rápida e intuitiva, permitiendo administrar recordatorios y actividades desde cualquier dispositivo.

## 🔮 Próximas mejoras

- Calendario mensual.
- Notificaciones automáticas.
- Filtros por prioridad.
